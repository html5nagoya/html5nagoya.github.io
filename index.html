概要

HTML5NAGOYA は HTML / CSS / JavaScript を始めとした Web のフロントエンドの技術全般を学び合う勉強会です。

発表者が発表する形式の勉強会や、各々がやりたいことを持ち寄って行うもくもく会、ハンズオン形式の勉強会など、様々な勉強会を開催しています。

お互いがアウトプットすることで共にスキルアップしていくことを目指しています。 皆さんのアウトプットで成り立っている会です。お誘い合わせのうえご参加ください！
